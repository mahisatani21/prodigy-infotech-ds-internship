# 🌳 Decision Tree Classifier – Bank Marketing Dataset

## 📌 Project Overview

This project builds a **Decision Tree Classifier** to predict whether a customer will subscribe to a product/service based on their **demographic and behavioral data**. The analysis is performed using the **Bank Marketing dataset** from the UCI Machine Learning Repository.

## 🎯 Objective

* Develop a classification model using Decision Tree
* Analyze customer data to identify patterns influencing purchase decisions
* Predict whether a customer will subscribe (`Yes/No`)


## 📁 Dataset

* **Dataset:** Bank Marketing Dataset (UCI)
* Contains information such as:

  * Age
  * Job
  * Marital Status
  * Education
  * Contact Type
  * Campaign Details
  * Previous Outcomes
  * Target Variable: Subscription (Yes/No)

## ⚙️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn
* Jupyter Notebook

## 🧹 Data Preprocessing

* Handled missing and inconsistent values
* Encoded categorical variables
* Feature selection and transformation
* Split dataset into training and testing sets

## 🌳 Model Building

* Implemented **Decision Tree Classifier**
* Trained the model on training data
* Tuned parameters to improve performance

## 📊 Evaluation

* Evaluated model using:

  * Accuracy Score
  * Confusion Matrix
  * Classification Report

## 📈 Results & Insights

* The model successfully predicts customer subscription behavior
* Certain features like **contact type, previous outcomes, and campaign duration** have strong influence
* Decision Tree provides easy interpretability of decision rules
