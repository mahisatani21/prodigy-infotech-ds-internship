# Data Visualization: Distribution Analysis

## Project Overview

This project focuses on visualizing the distribution of a variable using **bar charts** and **histograms**. The goal is to understand how data is distributed across categories (like gender) or ranges (like age).

## Objective

To create visual representations (bar chart or histogram) for:

* **Categorical data**
* **Continuous data** 


## Dataset

The dataset used in this project contains population-related data.
> Example variables:
* Age (continuous)
* Gender (categorical)
* Population counts

## Tools & Technologies

* Python 
* Pandas
* Matplotlib
* Jupyter Notebook

## Data Preprocessing
* Loaded dataset using Pandas
* Checked for missing values
* Cleaned and formatted data for visualization

## Visualization

### Histogram (Continuous Data)
* Used to visualize distribution of values like age
* Helps identify patterns such as skewness or spread

### Bar Chart (Categorical Data)
* Used to compare categories like gender
* Shows frequency/count of each category
